Design Reference

Cyran Labs — Émeraude · Ambre

Palette et tokens de référence pour l'app back-office (Next.js). Direction émeraude profonde + accent ambre, fonds neutres verdâtres, typographie éditoriale (serif titres / sans corps). À utiliser comme source de vérité pour générer l'UI.

02Typographie

Titres en serif éditorial (Georgia → web : Source Serif 4). UI/corps en sans (Calibri → web : Inter / system-ui). Code en mono (JetBrains Mono).
